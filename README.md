## ComputerVision_excercises

This repository contains a collection of hands-on exercises implemented as part of my self-study of Computer Vision and related mathematical foundations. The goal is to build a deeper understanding of core concepts by implementing them from scratch, rather than relying on high-level libraries.

### Topics covered

The planned exercises span a range of foundational topics, including:
- 2D and 3D geometric transformations, Matrix-based transformation pipelines
- Interactive geometry manipulation
- Image processing (to be extended)
- Convolution and filtering
- Edge detection, gradients
- Corner detection, SIFT
- Homography, Image stitching

### Notes
This is a learning-oriented repository, not a production-ready project. The code reflects an iterative process of experimentation and understanding rather than final optimized implementations.
